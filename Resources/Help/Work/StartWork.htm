Основное окно программы состоит из главного меню, предоставляющего доступ ко всем функциям программы, и нескольких таблиц. Над таблицами находится панель вкладок. Нажатие на вкладку отображает соответствующую таблицу для просмотра и редактирования.

Все таблицы обладают стандартным интерфейсом и поддерживают большинство базовых функций. С помощью мыши можно выделять ячейки, редактирование производится по двойному клику. Перемещаться по таблице можно с помощью прокрутки (в том числе колёсиком мышки), с помощью клавиш со стрелками, либо по нажатию (Shift+)Tab/Enter (в этом случае курсор выбирает ячейки для ввода, игнорируя ячейки, доступные только для чтения).

При перетаскивании границы между строками/столбцами изменяется их размер. Правая кнопка мыши открывает меню с основными командами: отмена и возврат действий, копирование, вставка, выделение и удаление данных. Таблицы поддерживают все стандартные клавиатурные сокращения (Ctrl+Z/Y, Ctrl+C/V, Ctrl+A и другие); корректно обмениваются данными через буфер с другими программами с табличным редактором (Calc, Excel, Statistica).

При запуске программа по умолчанию откроет пустой проект, не содержащий никаких данных. Его нельзя проанализировать или сохранить, это только отправная точка, из которой начинается работа с программой. Вы можете открыть существующий проект для его модификации, выбрав пункт меню "Файл → Открыть" (клавиатурное сокращение Ctrl+O), либо создать на основе пустого проекта новый. Для этого добавьте в проект данные с помощью меню "Правка" (см. об этом раздел Добавление данных в проект).

Замечание. Вы можете в любой момент начать новый проект. Для этого нужно вернуться к пустому проекту, выбрав пункт меню "Файл → Создать" (клавиатурное сокращение Ctrl+N). Если текущий проект был изменён, перед возвратом к пустому проекту программа предложит сохранить данные.
