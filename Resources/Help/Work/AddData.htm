Чтобы создать проект на основе пустого необходимо добавить в него данные. Все операции с наборами данных осуществляются в меню "Правка". Сначала следует добавить необходимое количество признаков с помощью пункта меню "Добавить признаки...". Признаки появляются в конце таблицы признаков и необходимым образом расширяют остальные таблицы.

Для каждого признака необходимо ввести его имя, которое будет отображаться в таблице результатов, а также стандартное среднеквадратическое отклонение.

Замечание. Дополнительно, напротив каждого признака существует графа с галочкой, выставляя и снимая которую (с помощью клавиши пробел или двойного клика мышью) можно включать и исключать признак из анализа, не удаляя его из таблицы. Таким образом можно быстро проводить анализ одних и тех же выборок относительно разных наборов признаков.

Далее следует добавить требуемое количество выборок с помощью пункта меню "Добавить выборки...". Выборки появляются в соответствующей таблице. Для каждой выборки программе потребуется имя, которое будет отображаться в таблице результатов, числовая метка выборки, предназначенная для идентификации групп на графике, объём выборки (количество индивидуальных измерений) и среднее по измерениям из данной выборки значение для каждого из анализируемых признаков.

На последней вкладке необходимо ввести внутригрупповую корреляционную матрицу. Эта матрица по определению симметрична и имеет единичную диагональ. Поэтому изменение диагональных ячеек запрещено, а изменение любой другой ячейки автоматически заполняет симметричную ей ячейку тем же значением. Таким образом, достаточно ввести только половину матрицы (верхнюю или нижнюю — всё равно).

После заполнения всех таблиц можно переходить к Анализу данных.

Маленькие хитрости. Часто один и тот же набор признаков используется для анализа различных выборок. Вы можете создать для этого набора шаблон, то есть файл с заполненной информацией о признаках (первая и третья вкладки) и пустой таблицей выборок. С помощью этого шаблона можно быстро проводить новые расчёты: для этого откройте шаблон и сразу же сохраните его под новым именем (пункт меню "Файл → Сохранить как...", клавиатурное сокращение Shift+Ctrl+S). После этого добавляйте в файл выборки и работайте с ним как обычно. Все изменения будут сохраняться в новом файле, а старый файл (шаблон) останется нетронутым и пригодным для дальнейшего использования.
