Для выполнения анализа необходимо выбрать соответствующий пункт из меню "Расчёт". Поскольку результаты анализа будут открыты в том же окне, программа предложит сохранить изменения перед продолжением. Кроме того, перед запуском анализа программа проверяет все исходные данные. Если среди данных будет обнаружено отсутствующее или некорректное значение (значение в числовой ячейке, которое программа не сможет проинтерпретировать как число), будет выдано соответствующее предупреждение и анализ будет отменён. При этом программа автоматически выберет в нужной таблице некорректную ячейку (в частности, откроет соответствующую вкладку). После исправления значения можно повторно запустить анализ.

Замечание. В анализ включаются только те признаки, которые отмечены галочкой в таблице признаков. Ячейки, относящиеся к другим признакам, не участвуют в анализе и, соответственно, не проверяются на корректность. В частности, их можно оставить пустыми или записать туда временные данные (пометки для дальнейшего редактирования) в произвольном формате.

Если все данные будут успешно собраны, запустится процесс анализа. Он может занять некоторое время (в зависимости от размера данных, обычно не более десятка секунд). В это время программа может не отвечать на действия пользователя, что, однако, не является признаком её зависания. После окончания анализа вкладки исходных данных будут закрыты и на их месте появятся таблицы результатов. Все таблицы доступны только для чтения, поскольку являются вычисленными на основе исходных данных и не должны изменяться вручную.

Канонический анализ (пункт меню "Расчёт → Канонический анализ", клавиатурное сокращение Ctrl+1).

Результатами канонического анализа являются три таблицы. Первые две описывают канонические переменные: собственное значение, связанное с данной переменной, вклад в общую изменчивость и коэффициенты переменной по вошедшим в анализ признакам — нестандартизованные (первая таблица) и стандартизованные (вторая). Для нестандартизованных коэффициентов выводится также аддитивная константа, центрирующая выборки вдоль данной оси по среднему значению. Третья таблица содержит координаты всех выборок в нестандартизованных осях, предназначенные для построения графика. Для удобства в таблицу также дублируются из исходных данных метки выборок, с помощью которых можно разделять выборки на группы. Автоматическое построение графика по этим данным не предусмотрено, однако его можно легко осуществить вручную с помощью любой программы анализа табличных данных (Calc, Excel, Statistica).

Внимание! Большинство табличных редакторов в качестве разделителя целой и дробной части использует запятую. При использовании точки график может быть построен с ошибками. Программа MultiCan по умолчанию использует именно точку. Для облегчения переноса данных в другие редакторы существует пункт меню "Расчёт → Сменить десятичный разделитель", при выборе которого все точки автоматически заменяются на запятые. Повторный выбор данного пункта возвращает точки.
